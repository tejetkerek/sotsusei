デバッグチェックリスト

1. 基本チェック

  • F12で開発者ツールを開く
  • Consoleタブでエラーを確認
  • 赤いエラーメッセージがないかチェック
  • ページが正常に読み込まれるか

2. JavaScriptチェック

  • 括弧が正しく閉じられているか
  • セミコロンが抜けていないか
  • 関数名のタイプミスがないか
  • イベントリスナーが正しく設定されているか

3. HTMLチェック

  • タグが正しく閉じられているか
  • id属性が重複していないか
  • 属性の値が正しく引用符で囲まれているか
  • 外部ライブラリが正しく読み込まれているか

4. Firebaseチェック

  • Firebase設定が正しいか
  • 必要なサービスが有効化されているか
  • インポート文が正しいか
  • ネットワーク接続が正常か
